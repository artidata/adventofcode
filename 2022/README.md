# adventofcode-2022

Using TypeScript to solve https://adventofcode.com/2022/ puzzles.

- Change directory to the year of 2022:

```bash
cd adventofcode/2022
```

- Install dependencies:

```bash
npm install
```

- Execute `ts-node` to the target day and problem number:

```bash
npx ts-node 1/1.ts
```
